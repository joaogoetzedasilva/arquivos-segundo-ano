HTML

1.O que é HTML?

HTML (hyper Text Markup Language) é a linguagem usada para estruturar páginas da web.

2.Estrutura básica do HTML

<!DOCTYPE html>
<html lang="pt-BR">
<head>
<meta charset="UTF-8">
<title>Minha Página</title>
</head>
<body>
<h1>Ola, Mundo!</h1>
</body>
</html>
